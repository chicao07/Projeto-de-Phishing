# Requisitos

## Funcionais
RF01 - O sistema deve ser capaz de permitir que o usuario insira URLs.
RF02 - Deve ser capaz de validar a URL inserida pelo usuario e informar se ela é válida ou não.
RF03 - Deve ser capaz de analizar a(s) URLs recebidas.
RF04 - Deve ser capaz classificar as URLs por pontuação.
  - Segura 0 % a 20 %;
  - Parcialmente segura 21 % a 40 %;
  - Suspeita 41 % a 60 %;
  - Muito suspeita 61 % a 80 %;
  - Perigosa 81 % a 100 %.
RF05 - Deve ser capaz de retornar uma resposta final ao usuario.
RF06 - E apresentar os motivos do porque a URL recebeu tal classificação.
RF07 - Deve ser capaz de armazenar as URLs recebidas em seu historico. 


## Não Funcionais
RNF01 - Tempo de resposta em menos de 2 segundos.
RNF02 - Possuir uma interface simples e intuitiva.
RNF03 - Deve possuir uma breve introdução de como inserir uma URL
RNF04 - Ter uma boa disponibilidade
RNF05 - Deve ser compativél com os principais navegadores
RNF06 - Deve ser responsivo
