# Arquitetura 

## Front-end:
  - HTML (HyperText Markup Language)
    - Uma linguagem de marcação que definirá a estrutura e o conteudo (Onde ficam o texto, botão, e o campo de texto).
  - CSS (Cascading Style Sheets)
    - É a estetica da página. Ele define as cores, fontes, alinhamentos, espaçamentos e garante que o site fique responsivo
  - JS (JavaScript)
    - É o cérebro do front-end. Criando a interatividade do sistema com o usuario. 

## Back-end:
  - Python
    - Escolhido como a linguagem principal do Back-End devido à sua robustez e ecossistema maduro para segurança da informação e análise de dados, o que facilitou na criação desse sistema.
  - Flask
    - Um micro-framework Python escolido pela sua simplicidade, flexibilidade e alta performance que faz com que a resposta ao usuario seja mais rapida e eficiente, funcionando como uma ponte que conecta o Front-End à inteligência do Python.
